# Project Context

## Overview
Moderni SaaS landing page sa dark temom i orange akcentom. Sadrži: hero, features (3 kartice), pricing (3 plana), testimonials, FAQ accordion i footer.

## Tech Stack
- HTML5 (semantic)
- CSS3 (jedan style.css, bez JS frameworka)

## Structure
- index.html — Glavna stranica sa svim sekcijama
- style.css — Globalni stilovi, responzivni layout i tema

## Recent Changes
